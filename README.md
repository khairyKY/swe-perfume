# SWE Marketplace

Full-stack marketplace app with a Node/Express backend and a Vite/React frontend.

## Structure

- backend/ Node.js API (Express, MongoDB, Jest)
- frontend/ Vite + React client (Tailwind)

## Quick start

1. Backend

- cd backend
- npm install
- npm run dev

2. Frontend

- cd frontend
- npm install
- npm run dev

## Tests

- cd backend
- npm test

## Notes

- API endpoints and example requests are in backend/postman.
